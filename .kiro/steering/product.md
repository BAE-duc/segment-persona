# Product Overview

i-MAP is a data segmentation and analysis application designed for Japanese users. The application provides interactive visualization and analysis tools for customer data, featuring:

## Core Features
- **Segment Creation**: Interactive tools for creating and managing customer segments
- **Persona Analysis**: Multi-instance persona popup windows for detailed customer analysis  
- **Data Visualization**: Charts including radar charts, heatmaps, positioning maps, and composition ratio graphs
- **Comparison Tools**: Side-by-side comparison tables and analysis
- **Multi-modal Interface**: Popup-based workflow with drag-and-drop functionality

## Target Users
- Japanese business users (interface is primarily in Japanese)
- Data analysts working with customer segmentation
- Marketing professionals analyzing customer personas

## Key Workflows
1. Product selection → Segment creation → Persona analysis
2. Data filtering and selection through modal interfaces
3. Visual analysis through various chart types
4. Comparison and positioning analysis

The application uses a sophisticated popup management system allowing multiple concurrent analysis windows with proper z-index management and drag-and-drop positioning.
